What is Steeplejack?

  • Steeplejack is an opinionated way of scaffolding your NodeJS applications. It's great for building microservices, writing reusable software and makes 100% test coverage really easy.
  • Instead of repeatedly writing the same things over and over again, use conventions. Routing, data manipulation and logging become easy and enjoyable.
  • Web sockets are first class citizens so you can make real time applications as easily as you can a RESTful or full stack project.
  • A simple and reliable dependency injection system allows you to write intelligent unit tests that can interrogate what happens.
  • Give it a go and fall in love with writing robust software again.
